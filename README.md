# Super_Store_sales_analysis_using_powerbi
This project presents an interactive Sales Performance Dashboard built using Microsoft Power BI to analyze and visualize key business metrics from a Superstore dataset. The dashboard provides a comprehensive view of sales, profit, cost, and unit performance across different dimensions such as segment, product, and geography.

The primary objective of this dashboard is to enable stakeholders to monitor overall business performance, identify high-performing segments, and uncover areas requiring improvement through data-driven insights.

# Key Highlights
Total Sales: 118.73M
Total Profit: 16.89M
Total Units Sold: 1.13M
Total COGS: 101.83M

# Dashboard Features
KPI Cards: High-level metrics for quick business overview
Segment Analysis: Profit distribution across Government, Small Business, Enterprise, etc.
Geographical Insights: Profit comparison across countries
Product Performance: Sales and profit trends by product categories
Interactive Filters: Year-based filtering for dynamic analysis

# The dashboard is designed to:
Track sales and profitability trends
Analyze performance across different customer segments
Identify top-performing products and regions
Support strategic decision-making using data insights

# Conclusion
This dashboard provides a centralized and interactive platform for analyzing Superstore sales data. By combining multiple performance indicators and visual insights, it helps businesses better understand their operations, optimize strategies, and improve overall profitability.
